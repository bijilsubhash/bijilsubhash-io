My work spans the full spectrum of the modern data stack - from ingestion and transformation to orchestration, governance, and infrastructure as code.

I specialise in designing and deploying end-to-end data platforms using tools like dbt, Dagster, dlt, BigQuery, Snowflake, Databricks, Terraform, and Python.
